Interactive Food Pyramid

Todays food intake

An interactive Food Pyramid for Web Information Processing

Requirements

The front end of an Interactive food diary that allows users to record the different food shelves consumed each day along with that days date. Only HTML elements were used to create the chart, no images were used. Colour and other styling were achieved using CSS. “Pure” Javascript was used only, JQuery or any other Framework was not allowed.

Main requirements: An indicator is provided showing the number of items consumed for a particular shelf. Buttons are included to increase or and decrease the number of items on the shelf which only appear when you hover over their shelf. Interactive element for inputting date for the daily record is provided below.

Extra credit: The height of the individual Food Pyramid Shelves are adjusted depending on the number of items on that shelf. The colour of the circle containing the number of items is updated depending on whether the number is within the recommended guideline quantities for that shelf which corresponds to the shelf level. The overall pyramid shape is preserved regardless of the height of the individual shelves.
